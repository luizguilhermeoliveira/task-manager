# task-manager
Um sistema para gerenciar tarefas pessoais, permitindo a criação, atualização, exclusão e listagem de tarefas, com suporte a prioridades e categorias.

# Dependencias do Projeto e sua Utilidade

- Spring Web: Para criar a API REST
- Spring Data JPA: Para o banco de dados
- H2 Database: Banco de dados em memória do projeto
- Validation: Para validar os campos
- Lombok: Para reduzir código repetitivo como Getters e Setters
